# 小程序启动指南

## 📱 如何在微信开发者工具中运行

### 步骤1: 安装微信开发者工具

如果还没有安装,请从官网下载:
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 步骤2: 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"或点击"+"号
3. 填写以下信息:
   - **项目名称**: 网球预定系统
   - **目录**: 选择 `d:\github\booksystemTongYi\miniprogram` 文件夹
   - **AppID**: 选择"测试号"或填入你自己的AppID
4. 点击"确定"

### 步骤3: 配置后端地址

确保后端服务正在运行:
```bash
cd d:\github\booksystemTongYi\server
go run simple_server.go
```

后端应该运行在: http://localhost:8080

### 步骤4: 编译运行

1. 微信开发者工具会自动编译项目
2. 如果看到错误,检查控制台输出
3. 点击"编译"按钮重新编译

### 步骤5: 测试功能

#### 登录页面
- 首次进入会显示登录页面
- 点击"微信授权登录"按钮(开发环境下会模拟登录)

#### 首页
- 显示场地列表
- 支持下拉刷新
- 支持上拉加载更多
- 点击场地卡片进入详情页

#### 场地详情
- 显示场地图片和信息
- 选择日期(最近7天)
- 查看可预定的时间段
- 点击"立即预定"按钮

#### 确认预定
- 显示预定信息摘要
- 点击"确认预定"完成订单

#### 我的预定
- 查看所有订单
- 按状态筛选(全部/待支付/已完成/已取消)

#### 个人中心
- 显示用户信息
- 访问我的预定
- 退出登录

## 🔧 常见问题

### 1. 网络请求失败

**原因**: 后端服务未启动或地址配置错误

**解决**:
- 确保后端服务在运行: `http://localhost:8080`
- 检查 `app.js` 中的 `baseURL` 配置

### 2. TabBar图标不显示

**原因**: 图标文件缺失

**解决**: 
- 暂时使用文字TabBar(已配置)
- 后续可以添加图标文件到 `images/` 目录

### 3. 页面空白

**原因**: 可能是API返回数据格式问题

**解决**:
- 打开调试器Console查看错误
- 检查Network面板的API请求

### 4. 真机调试

如需在手机上测试:
1. 点击开发者工具的"预览"按钮
2. 用微信扫描二维码
3. 注意: 需要将后端地址改为公网IP或域名

## 📝 开发提示

### 修改后端地址

生产环境需要修改 `miniprogram/app.js`:
```javascript
globalData: {
  baseURL: 'https://your-domain.com/api' // 改为实际域名
}
```

### 启用HTTPS

微信小程序要求:
- 正式环境必须使用HTTPS
- 域名需要在微信公众平台配置
- 开发阶段可以使用HTTP + localhost

### 调试技巧

1. **开启调试模式**:
   - 点击开发者工具右上角的"详情"
   - 勾选"不校验合法域名"

2. **查看日志**:
   - Console面板查看JavaScript日志
   - Network面板查看网络请求

3. **清除缓存**:
   - 工具栏 → 清缓存 → 全部清除
