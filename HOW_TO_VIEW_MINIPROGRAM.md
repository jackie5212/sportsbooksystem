## 📋 操作步骤 (5分钟完成)

### 第1步: 下载微信开发者工具

如果还没有安装:
- 下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 选择"稳定版 Stable Build"
- 下载安装后打开

### 第2步: 导入项目

1. 打开微信开发者工具
2. 点击 **"+"** 或 **"导入项目"**
3. 填写信息:
   ```
   项目名称: 网球预定系统
   目录: D:\github\booksystemTongYi\miniprogram
   AppID: 点击"使用测试号" (不需要真实AppID)
   ```
4. 点击 **"确定"**

### 第3步: 配置调试环境

导入后,在开发者工具中:
1. 点击右上角 **"详情"** 按钮
2. 选择 **"本地设置"** 标签
3. ✅ 勾选 **"不校验合法域名、web-view(业务域名)、TLS版本以及HTTPS证书"**
   - 这允许访问 http://localhost:8080

### 第4步: 编译运行

1. 等待自动编译完成(约10-30秒)
2. 如果看到错误,点击工具栏的 **"编译"** 按钮重新编译
3. 左侧模拟器会显示小程序界面

### 第5步: 测试功能

#### 登录页面
- 首次进入显示登录页
- 点击 **"微信授权登录"** 按钮
- 开发环境下会自动模拟登录成功

#### 首页 - 场地列表
- 看到3个网球场地的卡片
- 下拉可以刷新
- 上拉加载更多(演示数据有限)
- **点击任意场地卡片** → 进入详情页

#### 场地详情页
- 顶部图片轮播
- 场地名称、位置、描述、价格
- 日期选择器(今天起7天)
- 时间段网格(绿色=可预定,灰色=已预定)
- **选择一个时间段** → 底部按钮变亮
- **点击"立即预定"** → 进入确认页

#### 确认预定页
- 显示预定信息摘要
- **点击"确认预定"** → 提示成功,跳转到我的预定

#### 我的预定
- 底部Tab切换到"我的预定"
- 看到2条示例订单
- 顶部Tab可以筛选状态

#### 个人中心
- 底部Tab切换到"我的"
- 显示用户头像和昵称
- 功能菜单入口
- 退出登录按钮

## ❓ 常见问题快速解决

### 问题1: 编译错误 "module not found"
**解决**: 
- 检查文件路径是否正确
- 确保所有.js/.wxml/.wxss/.json文件都存在
- 重新编译

### 问题2: 页面空白
**解决**:
- 打开Console查看错误
- 检查Network面板的请求
- 确认后端服务在运行

### 问题3: TabBar不显示图标
**正常**: 当前配置为文字TabBar,不影响功能
**优化**: 后续可以添加图标文件

### 问题4: 网络请求失败
**解决**:
- 确认后端运行: http://localhost:8080/health
- 勾选"不校验合法域名"
- 检查防火墙设置
