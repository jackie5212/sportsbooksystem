# 小程序图标说明

## 需要准备的图标文件

请在 `miniprogram/images/` 目录下放置以下图标文件(尺寸建议: 81x81px):

### TabBar图标

1. **home.png** - 首页图标(未选中)
2. **home-active.png** - 首页图标(选中)
3. **booking.png** - 我的预定图标(未选中)
4. **booking-active.png** - 我的预定图标(选中)
5. **profile.png** - 个人中心图标(未选中)
6. **profile-active.png** - 个人中心图标(选中)

### 其他图标

7. **default-avatar.png** - 默认头像

## 临时解决方案

如果暂时没有图标,可以:
1. 使用微信开发者工具自带的默认图标
2. 从 https://iconfont.cn 下载合适的图标
3. 使用纯色背景+文字的简单设计

## 颜色建议

- 主色调: #07c160 (微信绿)
- 未选中: #999999 (灰色)
- 选中: #07c160 (绿色)
